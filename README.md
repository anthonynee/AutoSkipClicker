# Auto Skip Clicker 🚀

An intelligent automatic click tool that detects and clicks "Skip" buttons on your screen automatically.

No manual operation required. Just run the program and let it monitor your screen in real time.

## ✨ Features

- 🔍 Real-time screen monitoring
- 🎯 Automatic button detection using OpenCV template matching
- 🖱️ Automatic mouse movement and clicking
- ⚡ Fast response time
- 🔄 Continuous background monitoring
- 🖥️ Works with browsers and desktop applications
- 🛠️ Simple and lightweight
